# MindMatrixio

**Modern AI Tools Platform** — Next.js 15 + Tailwind + OpenRouter + Grok

Your all-in-one freemium AI toolkit for content, code, research & productivity.

## Features
- AI Chat (Grok + OpenRouter)
- Text Summarizer / Generator
- Code Assistant
- Modern Responsive UI
- Ready for Vercel deployment

## Tech Stack
- Next.js 15 (App Router)
- Tailwind CSS
- TypeScript
- OpenRouter + xAI Grok API

## Quick Start

```bash
git clone https://github.com/Gourav-512/MindMatrixio.git
cd MindMatrixio
npm install
npm run dev
```
